Aplicación desarrollada en C++ y Java que simula el modus operandi de una agencia de viajes,
haciendo uso de listas lineales y la entrada-salida de ficheros de texto para la lectura de documentos.

C++
Esta práctica nos permitirá familiarizarnos con C++ y la declaración
e implementación de clases, que entre sus elementos debe contener la forma canónica de la
clase. La forma canonica es la forma ortodoxa de declarar una clase en C++. Permite evitar
problemas de programación, y permite que una clase declarada de esta forma se pueda usar
de la misma manera que cualquier tipo tradicional de C. En concreto, las operaciones que
definen la forma canónica de una clase son:
- constructores
- destructor
- constructor de copia
- sobrecarga del operador asignacion (=) 

C++ y Java
Esta practica nos permitirá trabajar con la entrada-salida de ficheros de texto para la
lectura de documentos en C++ y Java, y poder extraer información relevante relativa a los
mismos. En particular, realizaremos:
1. Implementación y uso de entrada-salida de datos a partir de ficheros en C++ y Java.
2. Implementación de tipos de datos para el almacenamiento de la información leída
desde fichero en C++ y Java.
3. Implementacion de aplicaciones concretas con las clases definidas.
